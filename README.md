# FiveM-AdministrationPanel
An Administration Panel for FiveM Servers


### Features
* Warning
* Kicking
* Commending
* Temporary Banning
* Permanent Banning
* Playtime Tracking
* Trust Score System
* Indepth Web-UI
* Indepth API
* Custom Server Action Buttons
* Some Statistics
* Multiple Servers
* First Joined
* Last Played
* Add/Remove Staff Menu
* Add/Remove Server Menu
* Edit Panel Menu
* Warnings/Kicks/Bans Logging
* Advanced Permissions System
* Player IP Tracking/Viewing (Permission Based)
* In-Game Commands
* One-Click Unbanning
* Removing Records
* Recent Players
* Per Staff Statistics
* User Search
* Staff Statistics


### Requirements
* Web Server w/ PHP
* MySQL Database
* Steam API Key (https://steamcommunity.com/dev/apikey)
* Support for Cron Jobs
